# Security Policy
Welcome to the Security Policy, stay and see the options for reporting a problem to the ghascan team.

## Reporting a Vulnerability

You can report security issues related to **ghascan and its dependencies**.

### Remember that:

Misfunctions, bugs, code errors, and more should be reported in **Issues**.

We are currently unable to offer bounties because it's a small project, but it's perfect for Bug Hunters, AppSec, developers who want to keep their projects secure, etc.

## Thank you for your understanding.
